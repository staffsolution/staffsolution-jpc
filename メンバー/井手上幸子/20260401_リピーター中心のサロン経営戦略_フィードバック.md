# ■案件名：リピーター中心のサロン経営戦略についての考察

# ■日付：2026年4月1日

# ■対象者：井手上幸子

---

## ■分類（タグ）
- LP改善
- 思考ズレ
- その他：構成・因果関係

---

## ■現状（Before）
- リピーター95%・完全予約制・1人最大2〜3人という自社の強みを整理しようとしている
- 一般サロンとの差別化ポイントを列挙している
- 「なぜそうしているのか」の理由を入れようとしている

### 問題点
1. **問題の深掘りが不足**：予約状況が読めないことで「どんな問題が起きるか」まで掘り切れていない
2. **ベネフィットの深掘りが不足**：来店数が予測できることで「どんな良いことがあるか」まで書けていない
3. **重複が多い**：同じ内容が複数箇所に出てきている
4. **なぜの理由がない**：人数制限・ゆとりある施術など「なぜそうしているのか」の理由が抜けている

---

## ■問題の本質（重要）
- **「何をしているか」は書けているが「なぜそうしているのか」「それによってどうなるのか」が抜けている**
- 問題→結果、強み→ベネフィットの因果関係が繋がっていない

---

## ■ズレ（思考 or 設計）

### 構造のズレ
1. **問題の深掘りがない**
   - 「当日予約・直前予約が多い → 空きと詰まりがバラバラ」まで書いてある
   - でも「それによってどんな問題が起きるか」が抜けている
   - 例：スタッフの疲弊、施術の質低下、経営が不安定、など

2. **ベネフィットの深掘りがない**
   - 「来店ペースが決まっている → 来店数を予測できる」まで書いてある
   - でも「それによって何が良くなるか」が抜けている
   - 例：スタッフが余裕を持てる、施術の質が上がる、経営が安定する、など

3. **重複が多い**
   - 「95%リピーター中心」と「完全事前予約で埋まる」は同じ話
   - まとめて1箇所に整理すべき

4. **「なぜ」がない**
   - なぜ1人最大2〜3人に制限しているのか
   - なぜお客様にゆったりした時間を過ごしてもらいたいのか
   - なぜスタッフにゆとりを持って施術してもらいたいのか

---

## ■改善内容（What）

### 1. 問題の深掘りを追加
```
【一般サロンの問題】
当日予約・直前予約が多い
    ↓
空きと詰まりがバラバラに発生
    ↓
・スタッフが疲弊する
・施術の質が安定しない
・経営が不安定になる
・お客様に集中できない
```

### 2. ベネフィットの深掘りを追加
```
【当サロンの強み】
95%がリピーター・完全事前予約
    ↓
来店ペース・来店数を予測できる
    ↓
・スタッフが余裕を持って準備できる
・1人1人の施術に集中できる
・お客様にゆったりした時間を提供できる
・経営が安定する
```

### 3. 重複を整理
- 「95%リピーター」と「完全予約制で埋まる」を1つにまとめる

### 4. 人数制限の理由を追加
```
なぜ1人最大2〜3人に制限しているのか
→ 毎回の施術にしっかり集中するため
→ お客様1人1人に向き合うため
→ スタッフにゆとりを持って施術してもらうため
```

---

## ■改善理由（Why）
- 「なぜ」があると読んでいる人が納得できる
- 因果関係でつながると、強みが「すごい」と伝わる
- 重複を省くと、伝えたいことが明確になる

---

## ■改善後イメージ（After）

```
【一般サロンの課題】
当日・直前予約が多い
→ 空きと詰まりがバラバラ
→ スタッフが疲弊・施術の質が不安定・経営が安定しない

【当サロンが選んだ道】
95%リピーター・完全事前予約
→ 来店数を予測できる
→ スタッフが余裕を持って準備できる
→ 1人1人の施術に集中できる

【なぜ人数を制限しているのか】
毎回の施術にしっかり集中するため
お客様にゆったりした時間を過ごしてもらうため
スタッフにゆとりを持って施術してもらうため
```

---

## ■再利用パターン（超重要）
- **強みを書くときは「なぜ」と「それによってどうなるか」をセットで**
- 問題を書くときは「それによってどんな問題が起きるか」まで深掘る
- 同じ内容が複数箇所に出てきたらまとめる

---

## ■NGパターン
- 「何をしているか」だけで「なぜ」がない
- 問題を列挙するだけで結果まで書かない
- 強みを書いてもベネフィットまで繋げない
- 重複を放置する

---

## ■一言まとめ（本質）
**「何をしているか」「なぜそうしているか」「それによってどうなるか」の3点セットで書く**
