# 一般店との違いを因果で深掘りする（フィードバック）

2025-08-30

## 相談内容

業種に関する情報の整理と、他社との違いのドラフトへのフィードバック。「一般店のペイン → その結果どうなるか」の因果を深掘りし、当社との差を具体的に出していく方向の指示。

## 全体の評価

業種に関する情報はバッチリ。次は「他社との違い」のブロックを強化する。現状は「狭い・臭い・ベタベタ」という事実が並んでいるが、「その結果どうなるのか」までつなげることが重要。これを繰り返すことで、読み手がリアルに「あるある」と感じる情報になる。

## 因果を深掘りするときの問いかけ

各情報に対して「その結果どうなるのか？」を自問し続ける。これがリアリティを作る。

- 狭いキッチン → **その結果どうなるのか？** → 動きにくい・ぶつかる・精神的に消耗する
- 匂いが染み付く → **その結果どうなるのか？** → 帰宅後も体や髪に匂いが残る・家族に指摘される・着替えても取れない
- ハウスクリーニングを頼まない → **その結果どうなるのか？** → 汚れが蓄積する・より臭いがひどくなる・長期的に清潔感が失われる
- スタッフ人数が少ない（15〜18人 vs 25人以上） → **その結果どうなるのか？** → 一人あたりの負担が増える・休みにくくなる

この因果の流れをターゲット（主婦・大学生）別に落とし込む。同じ「匂いがつく」でも、主婦と大学生では「嫌だ」と感じる角度が違う。

## 各ポイントへの深掘り方針

### 狭いキッチン

一般店が投資を抑えるためにキッチンを狭く作る → 動線が悪い・物にぶつかる → 精神的・体力的に消耗 → 早期離職につながる

### 匂い・べたべた（主婦目線）

- 髪の毛に匂いがつく・体に染み付く・着替えても取れない
- 家族から「臭い」と言われる → 自尊心が傷つく
- 洗濯・料理など家庭での生活への影響が気になる

### 匂い・べたべた（大学生目線）

- バイト後に友人と会いづらい・遊べない
- 講義・サークル後に匂いが気になる

### 急な休み連絡はいつまで大丈夫か

「当日でも大丈夫」は主婦（子供の体調）・大学生（試験・体調不良）どちらにも刺さる。「前日まで？当日？最短いつ？」を明確にするとリアルな安心感が伝わる。

### トレーニング（初期研修30時間・専任トレーナー）

- 一般店：営業しながらトレーナーが教える → 余裕がなくてピリピリ → 新人は質問しにくい
- 当社：営業に入らず専任トレーナーが担当 → 丁寧・わかりやすく教えられる → 新人も気兼ねなく質問できる
- 30時間をステップ（ステップ1・2・3）に分けて何日間かを明示すると、「安心して始められる」が伝わる
- なぜそういうトレーニング体制にしたのかの理由も入れると深みが出る

### 声かけ・接客スキル（コミュニケーション習得）

- 大学生向け：面接・社会人として使えるコミュニケーション基礎が身につく
- 礼儀・言葉遣いの具体的な指導があるなら「他社との違い」として出す
- 一般的なお店ではほとんど教えてもらえないが、当社では○○している、という比較で出す

### 食事割引・賄い

いくらのメニューがどのくらい安くなるのか、どんなものが食べられるのかを具体的に。

---

## やりがちなミス

- 「狭い・臭い」など事実だけ並べて「その結果どうなるのか」をつなげずに終わる
- ターゲット（主婦・大学生）で分けずに一つの表現で全員に向けてしまう
- トレーニングの「30時間」という数字だけ出して、ステップの中身が伝わらない

## 他の案件でも使えるパターン

- 「その結果どうなるのか？」を自問することで、事実から読み手の感情・行動への橋渡しができる
- ターゲットごとに「嫌だと感じる角度」が違う。同じ情報でも主婦・大学生に分けて翻訳する
- 研修・トレーニングの情報はステップ・日数・場所・理由をセットで出すと「安心」が伝わる
