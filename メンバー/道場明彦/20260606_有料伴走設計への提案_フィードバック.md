# 有料伴走設計への提案（道場所感・運営返答）

2026年6月6日

6月5日JPCサポート定例の録画を拝見いただき、人事制度構築支援の実践知から所感をいただいた内容への返答と、商品設計への反映記録です。

---

## 道場さんからの所感（要約）

経営者が欲しいのは学ぶ場ではなく「成果」。成果物が得られる仕組みの方が価値を感じやすい、という整理。

人事制度構築支援の例：

- 実質は「内製化サポート」。コンサルが勝手に作るのではなく、会社に合ったものを一緒に作る
- **コンサル＋社長＋先方スタッフ**のチーム。月2回×1回3時間、MTGで手も動かす
- 決めるのは先方。ファシリは進捗管理・場の設定
- **社長も毎月MTGに同席**——②丸投げ社長でも巻き込む。不在だと後から「そんなターゲットは駄目だ」とひっくり返す

JPCメンバーはコンサル級スキルはないが、次の2つなら再現できるのでは、という提案：

1. **サポーター＋社長＋先方スタッフ**で手を動かす機会（MTG＝強制力）
2. **山本FB**（品質担保）

サポーターは進捗管理・場の設定・ファシリ。内容ジャッジまでは不要。

---

## 運営からの返答

6/5定例欠席、失礼しました。録画と所感、ありがとうございます。

方向性は6/5で固めた「社員任せ＋社長は確認・成果を見る」と矛盾しません。特に刺さったのは次の3点です。

**① 学ぶ場より成果物**

社長は結果しか見ない、という整理は商品の芯と同じです。LMS＝インプット、JLC＝アウトプット、という両方確認も、最終的には「求人が形になるか」が問いです。

**② 社長同席は必須**

社長＋社員だけの社内MTGは、いずれ薄れます。②丸投げ社長をMTGに入れないと、あとからひっくり返す——求人版のメンバー所感は、そのまま設計に入れます。「社長の代わりに伴走」はNG。社長同席＋外部が月1の場を設ける、が外向きの言い方です。

**③ サポーター＝ファシリ（内容ジャッジ不要）**

JPCメンバーのスキルレンジと200社上限の両方に合います。求人の内容ジャッジはJLC FB。必要なら山本が節目で見る、という分担です。

---

## 商品設計への反映（2026-06-06 確定）

所感を踏まえ、**有料伴走「月1の場」**を次のとおり確定しました（社内正本：`JLC/宣伝/2026年6月以降のプラン/新プラン/求人内製化アカデミー/18_実務サポートメニュー案.md` §6）。

| 項目 | 内容 |
|------|------|
| 名称 | 有料伴走／月1の場（「伴奏」ではなく**伴走**） |
| 価格 | **月29,800円**（回数を月2に増やしても同一料金） |
| 対象 | **従業員16名以上**を標準 |
| タイミング | 三ヶ月個別サポート終了後（4ヶ月目〜） |
| 参加者 | **サポーター＋社長（同席必須）＋担当社員** |
| 頻度 | **最低月1回**（60〜90分）。希望があれば**月2回まで** |
| 役割 | ファシリ・進捗管理・次の一手（内容ジャッジはJLC FB） |
| 価値 | 社内だけでは止まる。**外部が毎月「約束の場」を設ける** |

三ヶ月無料（初月2回→2月目1回→3月目希望者）の延長線上として位置づけています。

---

## 設計時に避けたいNG

- 社長不在で進める（あとからひっくり返しやすい）
- 社長＋社員だけの社内MTGだけに頼る（数ヶ月で薄れる）
- サポーターを「コンサル」として内容を決める（スキル要件が上がりすぎる）
- 「社長の代わりに伴走します」と伝える（丸投げを助長する）

---

## 他の案件・設計でも使えるパターン

**成果が見える第3層**

```
① インプット（LMS・研修）
② アウトプット（JLC・FB・事例）
③ 制作MTG（外部ファシリ＋社長同席＋担当者）← 止まらない枠
④ 節目の品質担保（山本FB等・回数限定）
```

**社長②（丸投げ）への対処**

内容を全部覚えさせるのではなく、**月1の決定MTGに同席**させる。不在OKにしない。
