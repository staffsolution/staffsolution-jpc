# 貸しスペース：キーワードグループ分けと構成（フィードバック）

2025年11月18日

**フィードバック動画：** [Loomで見る](https://www.loom.com/share/63c67fbc28394e9db0a80e4c5594b9e5)

## フィードバック本文

前回フィードバックを受けて、ターゲットを絞りすぎていた部分を調整。2つの言葉の組み合わせでキーワードを洗い出す方向に修正したというところまでの確認。

**キーワードの整理方針**

「事務所 選び方」など情報収集系のキーワードはコンバージョンにつながりにくいため外す。グループ化して整理する。

グループ構成の考え方：

| グループ | キーワード例 |
|---|---|
| 事務所賃貸系 | 事務所賃貸・事務所テナント・貸し事務所 |
| レンタル・シェアオフィス系 | レンタルオフィス・シェアオフィス（オフィスに絞ればレンタルもシェアも広く拾える） |
| クリニック・医療系開業 | クリニック開業・診療所賃貸 |
| 士業系 | 弁護士・税理士・会計事務所・社労士 など |
| エリア指定（岡崎）系 | 岡崎 貸し事務所・岡崎 レンタルオフィス など |

「社会保険労務士」「税理士」「会計事務所」などの士業系は別グループでまとめて整理する。

**エリア指定キーワードの扱い**

岡崎をつけたキーワードは検索ボリュームが非常に少ない：
- 岡崎 貸し事務所：30程度
- 岡崎 レンタルオフィス：710程度
- 岡崎 シェア必ず：20程度

ボリュームは少ないが、出しておけばクリック率が見込める。地域に絞ったキーワードはコンバージョン率が高い傾向があるため、ボリュームが少なくても除外せずに出す。

**2グループで網羅する戦略**

- ボリュームがある方：全国的な事務所・オフィス系キーワード → アクセスを集める
- ボリュームが少ない方：岡崎エリア指定キーワード → クリック率・コンバージョン率が高い

どちらも漏れなく出すことで、アクセス量と成約率の両面をカバーする。

**「事務所」がメインキーワードで最もボリュームが多い**

「貸しテナント」「テナント賃貸」も含めるが、メインは「事務所」系。「シェアオフィス」はオフィスに含まれているため、オフィスで配信すればシェアもレンタルも拾える。

---

## やりがちなミス

- 「事務所 選び方」など情報収集系をそのまま入れてクリックが増えても問い合わせにつながらない
- エリア指定キーワードをボリュームが少ないからと全部外してしまう
- グループを分けずにキーワードをフラットに並べてしまい、優先度が見えなくなる

---

## 他の案件でも使えるパターン

**ボリュームが少ないエリア指定キーワードも出しておく価値がある**

エリア指定キーワードは検索ボリュームが少なくてもクリック率・コンバージョン率が高い傾向がある。「どうせ少ないから」と除外せず、ボリューム系とエリア系の両方を出す設計にする。

**2グループ構成で「量」と「質」を両立する**

```
ボリューム系キーワード → 多くのアクセスを集める役割
エリア指定キーワード → 成約率の高いアクセスを集める役割
```

この2つを組み合わせることで、どちらか一方に偏らない広告設計ができる。

**コンバージョンにつながりにくいキーワードは最初から外す**

「〜の選び方」「〜とは」などの情報収集系は、クリックしても問い合わせにならないことが多い。広告費の無駄になるため、最初からリストに入れない。

## このフィードバックを学びに変える

- 事務所系・レンタル系・士業系・エリア指定系のグループで整理する
- 情報収集系キーワード（選び方・とは系）は除外する
- ボリューム系とエリア指定系の両方を出す設計にする
- 検索ボリュームと単価を確認してからメインキーワードを確定する
