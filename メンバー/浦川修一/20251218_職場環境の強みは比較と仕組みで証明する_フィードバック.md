# 職場環境の強みは比較と仕組みで証明する（フィードバック）

2025年12月18日

**フィードバック動画：** <a href="https://www.loom.com/share/f7fcf5f2c400493cbbc41d6a40609c63" target="_blank">Loomで見る</a>

---

## 修正点1：通勤時間は具体的な数字と他社比較で伝える

「地元中心の現場」という表現だけでは伝わらない。具体的な出発・帰宅時間を入れ、他社との差を数字で示す。

例：
```
当社：7時出発 → 18時過ぎに帰宅
他社：5時出発 → 20時過ぎに帰宅
往復で約2時間の差が生まれます。
```

「長時間移動の心配が一切ない」という言い切りに、この数字の根拠を添える。

---

## 修正点2：「相談しやすい環境」は会社の仕組みで証明する

「同世代・未経験スタートが多い」という事実だけでは、相談しやすい環境の証拠として弱い。

「相談しやすい」と書くなら、なぜそうなるのかを会社側の仕組みで示す。

整理の視点：
- 会社としてどんなルール・制度があるか
- パワハラや孤立を防ぐための仕組みがあるか
- 先輩スタッフがどう関わる設計になっているか

仕組みが特にない場合は、「平均年齢58歳・中途採用が多い・近続年数約9年」という事実を並べて、「自然に相談しやすい雰囲気が生まれている」という流れにする。

---

## 修正点3：平均勤続年数は業界比較とセットで伝える

「平均勤続年数約9年」という数字は、業界平均と比較することで初めて価値が伝わる。

例：
```
一般的にこの業界の平均勤続年数は約2年と言われていますが、
当社では約9年と、業界平均の4〜5倍以上長く働いている方が多くいます。
```

「長く働ける」という主張の根拠として、業界比較の数字を先に置く。

---

## 修正点4：福利厚生（むつみ会）はベネフィットと実際の声を添える

「むつみ会」という名称だけでは何か分からない。ベネフィットを先に書き、制度名はその後で説明する。

例：
```
月500円の会費で、年1回社員旅行に行けたり、各種お祝い金をもらえる制度に任意で参加できます。
（社員旅行は石川・富山周辺の温泉地へ一泊二日）
```

さらに以下を加えると説得力が増す：
- 旅行先・旅館の写真
- 実際に参加した社員の感想
- むつみ会に加入しない場合でも受けられる制度（結婚祝い・出産祝い・お見舞い金）の説明

---

## 修正点5：根拠の数字は必ず見直して正確にする

「他社と比べて2時間の差」「業界平均の4〜5倍」などの数字は、ずれていると信頼を失う。使う前に一度確認・見直しをする。

---

## やりがちなミス

- 「相談しやすい環境」と書いても、会社として何をしているかが見えない
- 平均勤続年数を出しても、業界平均との比較がないため価値が伝わらない
- 福利厚生の制度名だけを書いて、何が嬉しいのかが分からない
- 根拠の数字をチェックせずそのまま使う
- 通勤時間の有利さを「地元中心」という言葉だけで伝えようとする
