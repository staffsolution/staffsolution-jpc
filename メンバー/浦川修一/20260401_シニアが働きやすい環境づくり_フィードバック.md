# ■案件名：シニアが働きやすい環境づくりについての提案

# ■日付：2026年4月1日

# ■対象者：浦川修一

---

## ■分類（タグ）
- LP改善
- コピー
- 思考ズレ
- その他：構成・因果関係

---

## ■現状（Before）
- シニアが働きやすい環境・手取り月20万円以上・定年制度廃止などの強みを整理しようとしている
- 警備業界での差別化ポイントを書いている
- 無理な受注をやめたことで働きやすくなった経緯がある

### 問題点
1. **他社との比較が薄い**：「うちは手取り20万以上」だけでは差が伝わらない
2. **不安の先取りができていない**：「警備＝きつい」という先入観への反論がない
3. **因果関係が弱い**：「無理な受注をやめた → 急な呼び出しがなくなった」の流れが不明確
4. **正社員への誤解を解いていない**：「正社員＝融通が利かない」という思い込みへの説明がない
5. **今と昔の対比がない**：「昔はそうだったが、今は違う」という変化が伝わっていない

---

## ■問題の本質（重要）
- **読んでいる人の「不安・疑念」に先回りして答えていない**
- 強みを並べるだけで「でも、どうせ…」という反論が処理されていない
- 因果関係（なぜそうなったのか）がロジカルにつながっていない

---

## ■ズレ（思考 or 設計）

### 1. 他社との比較が弱い
- 「手取り月20万以上」と書いてあるが、他社ではなぜ達成できないのかが書かれていない
- 比較対象がないと「20万円以上」の価値が伝わらない

```
【追加すべき内容】
他の警備会社が月20万以上を達成できない理由
→ 支給額が少ない
→ 控除額が大きい（保険料等）
→ 勤務日数が少ない・安定しない
など
```

### 2. 「警備＝きつい」という先入観への反論がない
- シニアが警備に応募するとき、真っ先に感じる不安は「急な呼び出し」「連勤」「長時間勤務」
- その不安を先取りして解消しないと、強みが届かない

```
【流れのイメージ】
「月20万以上稼げる」
    ↓ でも読者の頭の中では…
「でも警備って急な呼び出しとかあるんじゃないの？」
「連勤や長時間勤務が続くんじゃないの？」
「それじゃ続けられない」
    ↓ だからこそ
「実は当社も昔は同じでした。でも今は違います」
```

### 3. 変化の因果関係が弱い
- 「無理な受注をやめた」という事実はあるが、なぜそれで働きやすくなったかの流れが不明確

```
【改善後の流れ】
無理な受注をやめた
    ↓
急な呼び出しがなくなった
    ↓
連勤・長時間勤務がなくなった
    ↓
シニアでも無理なく続けられる環境になった
```

### 4. 正社員への誤解を解いていない
- 「正社員＝勤務日数を自由に変えられない」という思い込みがある
- これを先回りして「勤務日数の調整は可能」と明示する必要がある

```
【追加すべき内容】
「正社員だと休みにくいんじゃないの？」
→ 勤務日数の調整が可能なので、安心してください
→ 週3日から始めてOK、などの具体例
```

### 5. 今と昔の対比がない
- 業界の「昔の当たり前」と「今の当社」を対比させると差が際立つ

```
【対比の例】
昔（業界の常識）→ 今（当社）
急な呼び出しあり → なし
連勤・長時間勤務 → 月24日・8時間勤務が基本
定年あり → 定年制度廃止
手取り不安定 → 月20万以上安定
```

---

## ■改善内容（What）

1. **他社比較を追加**：なぜ他社では20万以上達成できないのかを具体的に
2. **不安の先取り**：「警備＝きつい」という先入観に先回りして回答
3. **因果関係を整理**：無理な受注をやめた → 具体的にどう変わったかの流れ
4. **正社員の誤解を解く**：勤務日数調整可能であることを明示
5. **今昔の対比を追加**：業界の昔の常識 vs 当社の今

---

## ■改善理由（Why）
- 読んでいる人の不安に答えないと、いくら強みを書いても信じてもらえない
- 「でも、どうせ…」という反論を先に処理することで、信頼感が上がる
- ロジカルに因果関係でつながると、説得力が増す

---

## ■改善後イメージ（After）

```
【全体の流れ】

1. 問題提起（読者の不安の先取り）
「警備会社に応募したいけど、急な呼び出しや
長時間勤務が心配…そう思っていませんか？」

2. 共感（昔の業界の実態）
「実際、多くの警備会社では
そういった環境が今も続いています。
当社も、以前は同じでした。」

3. 転換（なぜ変わったのか）
「無理な受注をやめたことで、
急な呼び出しも連勤もなくなりました。」

4. 強みの提示（他社比較）
「他社では月20万以上の手取りが難しい理由：
・支給額が低い
・控除が大きい
・勤務日数が安定しない
当社は月24日勤務で手取り20万以上を実現。」

5. 誤解の解消
「正社員だから融通が利かないのでは？
→ 勤務日数の調整は可能です。」

6. まとめ
「シニアが無理なく、長く働ける環境を
整えています。」
```

---

## ■再利用パターン（超重要）
- **読者の不安・疑念を先回りして処理する**
- 強みは「他社と比べて何が違うか」で伝える
- 変化を伝えるときは「なぜ変わったのか」の因果関係を入れる

---

## ■NGパターン
- 強みだけ並べて、読者の不安に答えない
- 比較なしに「20万以上」と書いても価値が伝わらない
- 「変わりました」だけで、なぜ変わったかを書かない

---

## ■一言まとめ（本質）
**読者の「でも、どうせ…」という疑念を先回りして処理した上で、強みをロジカルに伝える**
