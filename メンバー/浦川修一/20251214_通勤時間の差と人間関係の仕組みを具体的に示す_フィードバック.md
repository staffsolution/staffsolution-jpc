# 通勤時間の差と人間関係の仕組みを具体的に示す（フィードバック）

2025年12月14日

**フィードバック動画：** [Loomで見る](https://www.loom.com/share/da6990f718814f02b54acf1b917a175f)

---

## 修正点1：通勤時間の差は数字で「激しく」見せる

「地元中心の現場」という表現だけでは差が伝わらない。他社との出発・帰宅時間を並べて、実際にどのくらい違うかを数字で示す。

構成の例：
```
他社の場合：朝5時出発 → 帰宅は20時過ぎ
当社の場合：朝7時出発 → 帰宅は18時過ぎ
往復で約2〜3時間の差が生まれます。
```

さらに「会社に戻って装備を返却してから帰る」などの工程がある場合、それも含めた帰宅時間を書くと、他社の過酷さがより際立つ。「そんなに遅くなるの？」と思わせることが重要。

---

## 修正点2：人間関係の情報は「会社の取り組み」として書く

「フラットな人間関係」「気を遣いすぎない職場」は業種全体の特徴として書くと弱い。その会社が何かしているから良い人間関係が生まれているという形にする。

整理の視点：
- 会社として人間関係のトラブルを防ぐためのルールや仕組みがあるか
- 他社でよく起きる人間関係の問題（例：年功序列、ベテランからの圧力など）と、自社がそれと違う理由を説明できるか

例：
```
警備の現場では、ベテランと新人の関係でトラブルが起きることがあります。
当社では年齢や経歴よりも「安全に仕事を終わらせること」を最優先にする文化があり、
立場に関わらず意見を言いやすい雰囲気があります。
```

「他社ではこういう問題があるが、当社ではこうしているから大丈夫」という対比が信頼を生む。

---

## やりがちなミス

- 「地元中心」と書いても、他社と比べてどのくらい楽かが伝わらない
- 通勤時間の差を数字で見せず、読み手が「ちょっと近いだけ」と思ってしまう
- 人間関係について「フラット」と書くだけで、会社として何をしているかが見えない
- 業種全体の特徴として書いてしまい、その会社の独自性が出ない
